## Corona Symptoms Detection Chatbot
DATA-228 Big Data Project to implement a chat to predict covid-19 symptoms using pretrained model.

## Tech Stack
- PySpark (With OpenJDK 11 and Apache Spark 3.3)
- Socket.io
- Pandas
- Flask
